# AngularJS Mvc Demo

Companion Repo for [AngularJS Mvc Demo](https://github.com/ZhouYanlang/Angular-demo)

# Instructions

兼容IE8+以及主流浏览器

# 编译

1. npm install

2. gulp 

# 监听文件变化

gulp watch

# 开启本地服务和浏览器

gulp serve

# 依赖
